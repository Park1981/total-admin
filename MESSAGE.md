## Git 원격 주소 갱신 (토큰 포함)
```bash
git remote set-url origin https://<YOUR_GH_TOKEN>@github.com/Park1981/total-admin.git
git push origin master
```

> `<YOUR_GH_TOKEN>` 자리에 GitHub Personal Access Token 입력 후 실행해줘.
